Below is a step‐by‐step guide to help you troubleshoot and resolve the 400 error when verifying phone numbers with Prelude’s API:

1. Understand the Error

A 400 (Bad Request) response means that the data sent to the API is either missing or not formatted as expected. Common culprits include:
	•	Invalid phone number format: Prelude requires phone numbers in E.164 format (e.g., +15555555555).
	•	Incorrect or missing parameters: This can include custom codes, template IDs, date formats, locale settings, or device information.
	•	Malformed JSON or headers: Ensure the JSON is properly structured and that the request includes all required headers (like Content-Type).

Before making any changes, check the full error response (it should include an error code and a human-readable message). This message will often point to the specific parameter causing the issue.

2. Verify Your Request Parameters

Carefully compare your request payload with Prelude’s API documentation. Double-check that:
	•	Phone Number:
It is in the proper E.164 format. For example, if a user enters “555-1234” you must convert it to something like +15551234.
	•	Verification Code:
The code you received from Prelude must be passed exactly as provided—check for extra spaces or mismatches.
	•	Additional Fields:
If the API requires fields like a template_id, locale, or device info, confirm that you’re including these exactly as documented.

Node.js Example (Express):

// Ensure you have installed express and node-fetch (or use native fetch in recent Node versions)
const express = require('express');
const fetch = require('node-fetch');
const bodyParser = require('body-parser');

const app = express();
app.use(bodyParser.json());

const PRELUDE_API_URL = 'https://api.prelude.so/verify';
const PRELUDE_API_KEY = process.env.PRELUDE_API_KEY; // set this in your Replit environment variables

app.post('/verify-phone', async (req, res) => {
  const { phone_number, code } = req.body;
  
  // Basic check: Ensure phone number is in E.164 format
  if (!phone_number.startsWith('+')) {
    return res.status(400).json({ error: 'Invalid phone number format. Use E.164 format (e.g., +15555555555).' });
  }
  
  try {
    const response = await fetch(PRELUDE_API_URL, {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${PRELUDE_API_KEY}`
      },
      body: JSON.stringify({
        phone_number: phone_number,
        code: code,
        // Include other required parameters here (e.g., template_id, locale)
      })
    });
    
    const data = await response.json();
    
    if (!response.ok) {
      // Log the full error from Prelude to help diagnose the issue
      return res.status(400).json({ error: data });
    }
    
    res.json({ message: 'Phone verified successfully!', data });
  } catch (error) {
    res.status(500).json({ error: error.message });
  }
});

app.listen(3000, () => {
  console.log('Server running on port 3000');
});

Python Example (Flask):

from flask import Flask, request, jsonify
import requests
import os

app = Flask(__name__)
PRELUDE_API_URL = 'https://api.prelude.so/verify'
PRELUDE_API_KEY = os.environ.get('PRELUDE_API_KEY')  # Set this in your Replit secrets

@app.route('/verify-phone', methods=['POST'])
def verify_phone():
    data = request.json
    phone_number = data.get('phone_number')
    code = data.get('code')
    
    # Basic check for E.164 format
    if not phone_number.startswith('+'):
        return jsonify({"error": "Invalid phone number format. Please use E.164 (e.g., +15555555555)."}), 400

    payload = {
        "phone_number": phone_number,
        "code": code,
        # Add any other required parameters, such as template_id or locale
    }
    headers = {
        "Content-Type": "application/json",
        "Authorization": f"Bearer {PRELUDE_API_KEY}"
    }
    
    response = requests.post(PRELUDE_API_URL, json=payload, headers=headers)
    data_response = response.json()
    
    if response.status_code != 200:
        return jsonify({"error": data_response}), 400
    
    return jsonify({"message": "Phone verified successfully!", "data": data_response})

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=3000)
